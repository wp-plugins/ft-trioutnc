== Description ==

This plugin creates a widget containing your most recent TriOutNC checkins. Options include displaying your checkins on a map or in a list, the number of checkins you would like displayed, and some basic styling options.

== Installation ==

1. Unzip & upload to the `/wp-content/plugins/ft-trioutnc` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Appearance -> Widgets and activate the widget.
